# my-obsidian

個人的な知識管理リポジトリ。プロジェクト関連のドキュメント、技術メモ、作業ログなどを集約しています。

## フォルダ構成

| フォルダ構成        | 役割                     | 公開 / 非公開 |
| --------------- | ---------------------- | -------- |
| docs/           | ドキュメント系                | 公開       |
| ├─ projects/    | プロジェクト別ドキュメント          | 公開       |
| ├─ tech/        | 技術関連のナレッジ・トラブルシューティング | 公開       |
| log/            | 作業メモ・個人用ログ             | 非公開      |
| template/       | 各テンプレートファイルを格納         | 公開       |
| Clippings/      | Web記事などのクリッピング         | 公開       |


## 使い方

このリポジトリはObsidianで管理することを想定しています。

1. Obsidianでこのフォルダを開く
2. 必要に応じて`template/`内のテンプレートを使用
3. プロジェクトドキュメントは`docs/projects/`に、技術メモは`docs/tech/`に整理
